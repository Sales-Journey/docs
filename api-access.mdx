## Requesting access

To request access to API Key and secret, your must have a Sales Journey account
enabled with a valid plan.

<Steps>
  <Step title="First Step">
    Access [**Sales Journey Platform**](https://app.salesjourney.io) with a valid account.
  </Step>

  <Step title="Second Step">
    On the top menu, click on <Icon icon="gear" size={16} /> **"Settings"**.
  </Step>

  <Step title="Third Step">
    Click on **"Integrations" menu**.
    *If this menu is not available, open a ticket to request the API Key and Secret
  </Step>

  <Step title="Fourth Step">
    Click on **"Access Keys"** and after on **"Request Access Key/Secret"**.
  </Step>

  <Step title="Fifth Step">
    Give a name to the Access Key/Secret.
    **Important**: Copy the Secret in a secure place, once the code is shown only on creation.
    If you lose or forget the Secret, request another one.
    This process disable all generated Secrets.
  </Step>

</Steps>
